I made this game for the fall 2025 OSU VGDC game jam. It is a 3D platformer with a custom gravity system.

I programmed the 3rd person camera and custom gravity based on the tutorials created by [catlikecoding](https://catlikecoding.com/unity/tutorials/).

The controls are WASD for movement, space for jump, left click for diving, and right click for gliding. 

The version I submitted to the game jam can be found on [my Itch.io page](https://wafflesthehutt.itch.io/).

I continued to work on the game afterwards, fixing multiple bugs and adding some additional mechanics, but I have sinced moved on to focusing on other projects. 
The gliding behaves very unusually in the version which is posted. I have removed gliding in my current game version and instead tuned the jumps to make gliding unnecessary.

On returning to this project I intend to create a system to visualize the gravity fields, but I will probably rebuild the project in Godot rather than continue it in Unity.
